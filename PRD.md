# 0731本地生活圈 - 产品需求文档 (PRD)

## 一、项目概述

### 1.1 基本信息
- **应用名称**：0731本地生活圈
- **网站类型**：功能服务型（社区平台）
- **产品类型**：本地生活社区
- **核心价值**：打造韶山人专属的实名认证本地生活圈，建立真实邻里信任关系
- **目标用户**：韶山本地居民（户籍/常住/工作人员）
- **品牌口号**：只服务韶山人，只做韶山事。每一个账号背后，都是你的邻居。

### 1.2 技术配置
- **数据库**：需要数据库（暂不启用，待页面生成后启用）
- **网站主语言**：中文
- **响应式要求**：适配移动端/平板/桌面，优先移动端体验

## 二、核心功能

### 2.1 用户认证系统
严格执行身份证+手机号双重实名认证，四级认证体系：
- 基础认证（手机号+身份证号）→ 浏览全站
- 标准认证（+人脸识别）→ 发帖/评论/点赞
- 高级认证（+户口/居住证）→ 交易/求职/拼车
- 商家认证（+营业执照）→ 发布商业信息

### 2.2 十大功能板块
1. 🔥 韶山新鲜事（热点资讯区）
2. 🍜 吃喝玩乐（美食娱乐区）
3. 🏥 便民服务（生活服务区）
4. 💻 数码科技（电脑/监控/音响专区）
5. 🛒 韶山集市（二手交易区）
6. 👷 求职招聘（工作信息区）
7. 🎓 教育亲子（家庭成长区）
8. 🏡 小区邻里（社区微网格）
9. 🚌 出行交通（出行信息区）
10. 💬 韶山茶馆（闲聊交友区）
11. 🔧 建言献策（民意直通区）

### 2.3 违规处理机制
- 首次违规：删帖警告
- 二次违规：禁言3天
- 三次违规：禁言30天
- 严重违规：永久封号+报警处理

## 三、页面结构

| 页面 | 作用 | 优先级 |
|------|------|--------|
| 首页 | 展示十大板块入口、热门帖子、社区公告 | 高（MVP） |
| 板块列表页 | 展示某板块下的所有帖子 | 高（MVP） |
| 帖子详情页 | 展示帖子完整内容、评论、互动 | 高（MVP） |
| 发帖页 | 用户发布新内容的表单页面 | 高（MVP） |
| 用户认证页 | 完成实名认证流程 | 高（MVP） |
| 个人中心 | 查看自己的帖子、收藏、消息 | 中 |
| 搜索结果页 | 展示搜索结果 | 中 |
| 帮助中心 | 解答疑问、社区准则说明 | 中 |

## 四、视觉策略

### 4.1 色彩方案
- **主色调**：暖橙色 #FF6B35
- **辅助色**：深蓝色 #1A3A5E
- **背景色**：米白色 #F9F6F2
- **卡片背景**：浅灰色 #E8E4DC
- **成功色**：绿色 #4CAF50
- **错误色**：红色 #F44336
- **警告色**：橙色 #FF9800
- **强调色**：金色 #D4AF37

### 4.2 设计语言
- 整体风格：温暖亲和的本地社区风
- 核心调性：真实、温暖、亲切、可信
- 布局模式：卡片式板块入口 + 传统线性信息流

### 4.3 响应式断点
- 移动端 < 768px：大按钮、单列布局、底部导航栏
- 平板 768px - 1023px：双列布局
- 桌面端 ≥ 1024px：完整功能、多列布局、侧边栏导航

### 4.4 交互动效
- 过渡时长：200-300ms（功能反馈），400ms（页面切换）
- 缓动函数：cubic-bezier(0.4, 0, 0.2, 1)
- hover效果：位移2px，缩放1.02

## 五、安全与隐私
- 所有证件照片加密存储
- 身份证号等敏感信息脱敏显示
- 联系方式由发帖人自行决定是否公开
- 异常登录自动冻结并通知用户
